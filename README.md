# AI Recommendation System

A simple recommendation system that suggests movies based on user preferences.

## Features

- Takes user input
- Matches preferences using logic
- Displays recommendations
- Easy to understand and implement

## Technologies Used

- Python

## How to Run

```bash
python recommendation_system.py
```

## Example

Input:
action

Output:
John Wick
Mad Max: Fury Road
The Dark Knight

## Concepts Used

- Pattern Matching
- Conditional Logic
- Recommendation Systems
- User Preferences
